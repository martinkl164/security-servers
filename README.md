# Security Servers

A collection of intentionally simplified demo systems (for example: banks) used to practice spec-driven delivery.

This repo is built around the [OpenSpec](https://github.com/Fission-AI/OpenSpec) workflow.

## Safety (LegitBank)
This is **not a real bank** and includes **no real authentication, payments, or banking actions**. Do not use this project to facilitate wrongdoing.
